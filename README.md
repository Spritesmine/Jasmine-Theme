# Jasmine-Theme

该仓库存放的 ServerStatus-Jasmine 默认的前端源码

## 第三方调用
 
* 一言接口
* 天气接口
* ip接口（页面底部ip显示）
* 腾讯地图接口（仅调用获取访客ip地点使用）
* 第三方随机图片接口（如有失效请及时反馈，谢谢）
